# 统计键盘敲击次数
### 1. 项目介绍
本项目是一个统计键盘敲击次数的小工具，可以统计键盘上每个键的敲击次数，以及每个键的敲击频率。
使用技术：Golang + Vue + ElementUI
### 目录说明
//(conf%2Fapp.conf)
(controllers)
(key_event)
(keys)
(node_modules)
(static)
(views)
(go.mod)
- `conf`：配置文件
- `aboutHtml`：关于页面
- `controllers`：控制器
- `key_event`：前端页面Electron主进程
